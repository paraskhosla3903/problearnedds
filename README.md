# Probabilistic Data Structures Overview

### Benefits

- Gives approximate answers using only sublinear space
- Time optimisation over exact deterministic solutions

### Usage and Applications

Natural applications characterise situations when space and time are both crucial and critical, 
and correctness of the solution is only so much important. If we can entertain some error in the solution
but not taking too much time or space, then Probabilistic Data Structures are the choice to make.

- Approximate Membership Queries (AMQ)
- Frequency Estimation Sketches
- Cardinality Estimators
- Similarity and Locality-Sensitive Hashing
- Random Sampling Algorithms - Reservoir Sampling
- Randomised Search & Sorted Structures
